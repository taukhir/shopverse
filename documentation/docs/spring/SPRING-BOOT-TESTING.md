Spring Boot testing is split into JUnit basics, Mockito, Spring MVC/repository tests, integration tests, coverage, and reliability practices.

| [JUnit Testing Fundamentals](testing/JUNIT-TESTING-FUNDAMENTALS.md) | Test pyramid, dependencies, JUnit annotations, lifecycle, structure, assertions, and parameterized tests. |

| [Spring MVC Repository And Security Tests](testing/SPRING-MVC-REPOSITORY-SECURITY-TESTS.md) | Controller tests, MockMvc, repository tests, Spring test annotations, context loading, and security tests. |
| [Integration Tests And Testcontainers](testing/INTEGRATION-TESTCONTAINERS.md) | Integration testing, Testcontainers, transactional tests, Kafka integration testing, and end-to-end testing. |
| [Coverage And Test Quality](testing/COVERAGE-TEST-QUALITY.md) | Coverage strategy, JaCoCo, other quality tools, exclusions, and testing do/do not guidance. |
| [Async Contract And Flaky Test Prevention](testing/ASYNC-CONTRACT-FLAKY-TESTS.md) | Async assertions, contract testing, test data, flaky-test prevention, parallel execution, guidelines, and references. |
